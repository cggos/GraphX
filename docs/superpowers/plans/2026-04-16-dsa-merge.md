### Task 3: Fix CMakeLists.txt tree path

**Files:**
- Modify: `dsa/CMakeLists.txt`

Since `tree/` is now at `../tree` relative to `dsa/`, two lines need updating.

- [ ] **Step 1: Edit include_directories**

In `dsa/CMakeLists.txt`, replace:
```cmake
include_directories(${PROJECT_SOURCE_DIR})
```
with:
```cmake
include_directories(${PROJECT_SOURCE_DIR} ${PROJECT_SOURCE_DIR}/..)
```

- [ ] **Step 2: Edit aux_source_directory for tree**

Replace:
```cmake
aux_source_directory(${PROJECT_SOURCE_DIR}/tree SRC_TREE)
```
with:
```cmake
aux_source_directory(${PROJECT_SOURCE_DIR}/../tree SRC_TREE)
```

- [ ] **Step 3: Verify the build succeeds**

Expected: all targets build without errors (`linked_list`, `binary_tree`, `sparse_matrix`, `algorithm`, etc.)

- [ ] **Step 4: Run tests (if GTest available)**
